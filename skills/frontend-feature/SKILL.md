# Skill: Frontend Feature

Use this skill for frontend pages, components, forms, state, and API integration.

## Steps

1. Identify route or feature area.
2. Read UI conventions and API contracts.
3. Separate server state from client UI state.
4. Define loading, empty, error, and success states.
5. Keep components aligned with the project design system.
6. Verify responsive behavior and text fit.

## Output sections

- Route/component scope
- API dependencies
- State model
- UI states
- Accessibility checks
- Tests or visual checks
