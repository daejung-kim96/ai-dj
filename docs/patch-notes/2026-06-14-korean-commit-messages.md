# 2026-06-14: Korean Commit Messages

## Summary

Added a default rule to write commit message summaries in Korean.

## Why

The user wants Git history to be easier to scan and understand while studying and evolving ai-dj.

## Rule

Use this format by default:

```text
type(scope): 한국어 요약
```

The conventional commit type and scope may stay in English for structure, while the summary should be Korean.
