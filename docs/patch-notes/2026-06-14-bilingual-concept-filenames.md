# 2026-06-14: Bilingual Concept Filenames

## Summary

Renamed study concept files to include both English and Korean names.

## Why

The user wants study materials to be easier to scan from filenames, not only from document titles.

## Rule

Use this filename format for study concept docs when practical:

```text
english-name--한국어-이름.md
```

Keep `README.md` as-is so GitHub renders the folder overview automatically.
